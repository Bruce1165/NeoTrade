## Dashboard 2.0 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **构建**: Vite (快速，现代)
- **UI**: Ant Design 或 Tailwind CSS
- **状态管理**: Zustand 或 React Query
- **图表**: ECharts (沿用)

### 后端
- **保持现有**: Flask API 不变
- **仅做前端**: React SPA 调用现有 API

### 部署
- **静态托管**: Vercel 或 Netlify (免费，全球CDN)
- **无需 ngrok**: 直接部署到公网

## 功能清单

### 核心功能（保留）
1. 筛选器列表展示
2. 运行筛选器
3. 检查结果（单股票检查）
4. 结果查询和下载

1. **实时日志**: WebSocket 推送筛选进度

3. **移动端适配**: 响应式设计
4. **主题切换**: 暗黑/亮色模式
5. **用户偏好**: 保存常用筛选器
