## 一、工作方法学习

### 变更影响检查清单法
**来源：** Bruce 反馈
**核心：** 每次调整逻辑时，必须列出所有可能受影响的环节，逐个检查

**应用示例（银凤皇信号三变更）：**
| 环节 | 是否受影响 | 检查状态 |
|------|-----------|---------|
| 核心筛选逻辑 | ✅ 是 | ✅ 已更新 |
| 日志输出描述 | ✅ 是 | ✅ 已更新 |
| 输出字段名 | ❌ 否 | ✅ 无需改 |
| 字段中文映射 | ✅ 是 | ✅ 已更新 |
| 前端 FIELD_ORDERS | ❌ 否 | ✅ 无需改 |
| 前端字段映射 | ❌ 否 | ✅ 无需改 |
| 输出文件格式 | ❌ 否 | ✅ 无需改 |
| 数据库记录 | ❌ 否 | ✅ 无需改 |

**记忆点：** 先列清单 → 逐个检查 → 确认无误 → 再测试

## 二、量化多头策略框架学习

### 6种主流策略与筛选器对应

| 策略 | 核心逻辑 | 对应筛选器 | 特点 |
|------|---------|-----------|------|
| **大催化剂** | 主题情绪博弈，龙头抱团 | *待开发* | 情绪溢价+流动性 |
| **中催化剂** | 事件/公告驱动，2-3周兑现 | *待开发* | 预期差+资金涌入 |
| **咖啡杯** | 横盘突破，放量启动 | ✅ 咖啡杯形态 | 惯性上涨+资金接力 |
| **量价爆发** | 量价格局，资金意图 | ✅ 二板回调、涨停倍量阴、双首板 | 盘口交易，偏短 |
| **聚焦最强主线** | 板块轮动，龙头溢价 | *待开发* | 行业β，只做当前热点 |
| **强势股回调低吸** | 趋势向上，急跌企稳 | ✅ 金凤皇、银凤皇 | 最主流，3-8天 |

### 20天周期铁律
1. 必须有短期催化（资金/情绪/事件/技术至少一个）
2. 不做深度价值（基本面不会大变）
3. 止损刚性（错了必须走）
4. 只做顺势/企稳（不猜底，不硬扛）

### 实用选择指南
- **想稳** → 强势股回调低吸（金凤皇、银凤皇）
- **想快** → 突破追强（咖啡杯形态、20日突破）
- **想确定性高** → 事件/公告驱动（*待开发*）

### 数据缺口识别
**已有：** 价格、成交量、涨跌幅、换手率

**缺失：**
- 热点主题数据源
- 公告/业绩/政策事件库
- 板块资金流向
- 龙虎榜/机构持仓
- 主题/概念标签

**结论：** 催化剂筛选器开发需先解决数据源问题

## 四、突破策略实战框架（20天周期）

### 核心定义
- **只做**：箱体/平台/前高放量突破
- **周期**：3-15天，最长20天
- **核心**：真突破拿，假突破砍

### 真突破入场条件（5必须）

| 条件 | 具体要求 | 数据支持 |
|------|---------|---------|
| **结构** | 横盘≥7天，突破前高/箱体/年线 | ✅ 可用价格数据 |
| **量能** | 成交量≥5日均量1.5-2倍，<3倍 | ✅ 有成交量数据 |
| **K线** | 收盘站稳突破位，中阳/大阳 | ✅ 可计算 |
| **分时** | 放量拉升，不破均价，午后不跳水 | ❌ 无分时数据 |
| **板块** | 板块强于大盘，有联动 | ❌ 无板块数据 |

**数据缺口：** 分时质量、板块强度

### 假突破识别（5不看）

1. 长上影
2. 次日低开低走
3. 爆巨量不涨停
4. 尾盘砸回突破位
5. 板块走弱，个股硬拉

**可检测：** 1、2、3、4（日线数据）
**不可检测：** 5（需板块数据）

### 买卖纪律

| 环节 | 规则 |
|------|------|
| **入场** | 突破当天尾盘/次日回踩不破 |
| **首仓** | 15-20% |
| **止盈1** | +8-12% 减仓一半 |
| **止盈2** | +15-20% 全部离场 |
| **时间止** | 10天不达标减半，20天强制清 |
| **止损** | 跌破突破位或-5%无条件砍 |

### 仓位规则
- 大盘多头：50-70%
- 震荡市：30-40%
- 下跌趋势：不做突破，空仓

### 与现有筛选器关联

| 策略要素 | 对应筛选器 |
|---------|-----------|
| 横盘突破+量能 | 20日突破（可升级） |
| 涨停+量能+回调 | 金凤皇、银凤皇 |
| 量价爆发 | 二板回调、涨停倍量阴 |
| 分时质量 | ❌ 无法判断 |
| 板块强度 | ❌ 无法判断 |

### 20日突破升级方向
1. 横盘天数检测（≥7天）
2. 量能条件精细化（1.5-2倍，防>3倍暴量）
3. K线实体检测（中阳/大阳）
4. 假突破过滤（长上影、次日低开）
5. 突破位跟踪（止损计算）

## 五、A股20天策略失败形态识别

### 一、假突破（最坑）

**特征：**
- 放量突破前高/箱体
- 当天冲高回落，收长上影
- 次日直接低开低走

**识别信号：**
- 量能异常暴量（>均量3倍）
- 突破后15-30分钟不封板/不强势
- 尾盘砸回突破位下方

**结论：** 诱多出货 → 不买，已买直接走

**数据可检测：** ✅ 量能、K线形态、次日开盘

---

### 二、放量长上影（避雷针）

**特征：**
- 盘中大涨，尾盘跳水
- 成交量巨大，K线像针

**含义：**
- 上方抛压极重
- 资金逢高跑路

**信号：** 见此形态，24小时内必减仓

**数据可检测：** ✅ 上影线长度、成交量、收盘位置

---

### 三、缩量阴跌式"企稳"

**特征：**
- 跌了几天，小阳线
- 成交量越来越小
- 不反弹、不拉升

**含义：**
- 没有资金进场
- 只是跌不动，不是要涨

**信号：** 阴跌中继，不是底

**数据可检测：** ✅ 成交量趋势、价格走势、反弹力度

---

### 四、均线空头排列下的反弹

**特征：**
- 5日<10日<20日均线
- 突然拉一根阳线

**含义：**
- 下跌趋势中的反抽
- 大概率一日游

**信号：** 只可超短快进快出，不可格局

**数据可检测：** ✅ 均线排列、阳线实体

---

### 五、高位高换手不涨停（死亡换手）

**特征：**
- 股价高位
- 换手率>20%甚至30%+
- 就是不封涨停

**含义：**
- 主力对倒出货
- 筹码大幅松动

**信号：** 见顶概率极高，次日不冲高就跑

**数据可检测：** ✅ 换手率、涨幅、价格位置

---

### 六、钓鱼线（早盘急拉午后走低）

**特征：**
- 开盘快速拉高3-7%
- 之后全天单边下跌
- 收盘接近最低价

**含义：**
- 标准钓鱼出货

**信号：** 出现当天就是离场点

**数据可检测：** ⚠️ 部分可检测（开盘涨幅、收盘位置），缺分时细节

---

### 七、利好兑现高开低走

**特征：**
- 政策/业绩/公告利好
- 大幅高开
- 低走收阴

**含义：**
- 利好出尽
- 资金借利好跑路

**信号：** 高开低走 = 最强空头信号

**数据可检测：** ⚠️ 可检测高开低走，但缺公告/利好数据源

---

### 失败形态可检测性总结

| 形态 | 可检测 | 关键指标 |
|------|--------|---------|
| 假突破 | ✅ | 量能>3倍、长上影、次日低开 |
| 放量长上影 | ✅ | 上影线/实体比、成交量 |
| 缩量阴跌 | ✅ | 成交量趋势、反弹力度 |
| 均线空头反弹 | ✅ | 均线排列、阳线实体 |
| 死亡换手 | ✅ | 换手率>20%、不涨停、高位 |
| 钓鱼线 | ⚠️ | 开盘涨幅、收盘位置（缺分时） |
| 利好兑现 | ⚠️ | 高开低走（缺公告数据源） |

### 筛选器应用建议

**可开发为过滤条件：**
1. 假突破过滤 → 加入20日突破筛选器
2. 死亡换手识别 → 独立预警筛选器
3. 缩量阴跌排除 → 在低吸策略中使用
4. 均线空头过滤 → 所有策略通用

**需数据源支持：**
- 钓鱼线精确识别 → 需分时数据
- 利好兑现识别 → 需公告/事件数据
