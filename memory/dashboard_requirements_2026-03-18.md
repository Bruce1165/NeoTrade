### 遇到的问题与修复

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| View Results 白板 | 表格渲染嵌套对象失败 | 创建动态 ResultsTable 组件，自动扁平化对象 |
| K线图不显示 | 弹窗未渲染完就初始化 ECharts | 添加 300ms 延迟 + 容器尺寸检查 |
| 列名显示英文 | 后端返回大写/下划线前缀字段 | 添加全大写版本翻译 + extra_data.xxx 映射 |
| Excel 格式不对 | 使用 .xls 后缀和旧 MIME 类型 | 改为 .xlsx + 正确的 HTML 格式 |
| 下载 CSV 而非 Excel | 代码实现的是 CSV | 重写为 HTML table 格式导出 |
