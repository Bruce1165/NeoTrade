# 2026-03-12 工作日志

## 杯柄形态筛选任务 (12month-cuphold)

### 任务创建
- 创建了可复用的Cron任务 `12month-cuphold`
- 运行命令: `/cron run 12month-cuphold`
- 输出文件: `data/12month_cuphold_YYYY-MM-DD.xlsx`

### 筛选条件演进

**初始条件 (10只股票):**
- 换手率 > 5%
- 收盘价涨幅 > 2%
- 放量倍数 1.5x - 2.0x
- 杯柄日期范围: 45-90个交易日前
- 最高价在昨日最高价的 0.9-1.1 倍之间

**结果:** 10只股票
- 芯能科技、中国电建、天富能源、泛微网络、禾望电气
- 大元泵业、西典新能、中盐化工、金煤科技、中闽能源

### 新增约束条件

**新增条件6:** 过去20-45个交易日股价 <= 昨日收盘价 * 0.85

**结果变化:** 从10只减少到 **1只**

**唯一入选:** 宁波建工 (601789)
- 放量倍数: 4.92x
- 换手率: 48.45%
- 杯柄日期: 2025-03-21
- 约束区间最高价: 5.46 (低于价格上限 6.21)

### 落选原因分析

其余9只股票全部因条件6落选:
- 在2025年12月底到2026年1月期间都有过上涨
- 约束区间最高价超过了昨日收盘价的0.85倍
- 例如泛微网络: 最高价74.51 >> 价格上限39.08
