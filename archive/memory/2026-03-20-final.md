## 🔧 Key Fixes Applied

### Date Handling (Critical)
- **Problem:** HTML5 date inputs showing localized format (2026/03/20) causing validation errors
- **Solution:** 
  - All dates normalized to `YYYY-MM-DD` via `toISODate()` function
  - State initialization uses `d.toISOString().split('T')[0]`
  - Removed all `getYesterday()`/`getToday()` calls from initial state

### Daily Hot/Cold Screener Bug
- **Problem:** Always returned 0 results for historical dates
- **Root Cause:** `df.iloc[-1]` always got latest row, not requested date
- **Fix:** Changed to `df[df['date_str'] == latest_date]` for date-specific filtering

### Backend API
- **Problem:** Hardcoded check blocked "today" from running
- **Fix:** Replaced with dynamic data availability check
